# ChordMaker Source Code

This directory contains the main application source code.

## Directory Structure

```
src/
├── components/      # Reusable React components
├── pages/          # Page components
├── utils/          # Utility functions
├── hooks/          # Custom React hooks
├── data/           # Data access layer
├── styles/         # CSS/SCSS files
├── App.js          # Main App component
└── index.js        # Entry point
```

## Key Files

- **App.js** - Main application component with routing
- **index.js** - React DOM render entry point

## Component Organization

- **ChordDiagram** - Interactive guitar chord visualization
- **ChordSearch** - Search and filter interface
- **ChordLibrary** - Chord reference library
- **ChordProgressions** - Common chord progression patterns
- **FretboardVisualization** - Interactive fretboard display
